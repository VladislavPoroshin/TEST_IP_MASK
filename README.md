# Инструмент для вычисления минимальной общей подсети двух IP-адресов

Данный проект реализует консольный инструмент для расчета наименьшей подсети, содержащей два заданных IP-адреса (IPv4 или IPv6). Программа парсит адреса, определяет общий сетевой префикс и выводит информацию о подсети в консоль.

## Особенности

- Поддержка **IPv4** и **IPv6** (включая сжатые формы `::`, зоны интерфейса `%`, встроенные IPv4-адреса).
- Парсинг адресов как без префикса, так и с **CIDR-префиксом** (`/24`) или **маской подсети** (`255.255.255.0`).
- Вычисление **минимальной общей подсети** – префикса максимальной длины, общего для обоих адресов.
- Форматирование результата:
  - версия адреса,
  - адрес сети (в каноническом виде),
  - длина префикса (CIDR),
  - маска подсети,
  - шестнадцатеричное представление сети.
- Обнаружение и обработка ошибок:
  - некорректный формат адреса,
  - недопустимые значения октетов,
  - неправильная маска подсети (не непрерывная или с октетами вне диапазона),
  - префикс вне допустимого диапазона,
  - недопустимые символы в зоне IP-адреса,
  - более одного `::` в IPv6,
  - несоответствие версий IP-адресов.
- Тесты (`test.py`) покрывают парсинг, вычисления и форматирование.

## Запуск

### Требования

- Интерпретатор **Python 3.6** или выше.
- Стандартная библиотека (модули `re`, `typing`, `sys`).

### Инструкция

Перейдите в следующий репозиторий:

```bash
cd task
```

#### Основная интерактивная программа

```bash
python main.py
```

Пример вывода:

```text
=== Вычисление минимальной общей подсети для двух IP-адресов ===

Введите первый IP-адрес: 192.168.1.1
Введите второй IP-адрес: 192.168.2.1

Результат:
  Версия протокола:      IPv4
  Адрес сети:            192.168.0.0
  Длина префикса (CIDR): /22
  Маска подсети:         255.255.252.0
  Hex представление сети: C0A80000
```

#### Тесты

```bash
python test.py
```

При успешном выполнении будет выведено количество пройденных и упавших тестов.
